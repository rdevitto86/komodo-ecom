# komodo-auth-api

OAuth 2.0 authorization server. Issues and validates RS256 JWTs for M2M (`client_credentials`) and user flows.

See [`docs/README.md`](./docs/README.md) for the full route reference, environment variables, middleware stack, and cURL examples.

| Key | Value |
|-----|-------|
| Port | 7011 (public), 7012 (internal) |
| Domain | Identity & Security |
| Status | Active |
